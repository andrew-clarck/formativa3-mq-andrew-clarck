# GalaxyTours

Este é um site da maior agência de turismo espacial do mundo.

Projeto realizado para a formativa de media query da aula de Front-End.

## Tecnologias Utilizadas

- HTML5 (Semântica e Acessibilidade)
- CSS3 (Estilização e Feedback Visual)
- JavaScript (Manipulação de DOM e Eventos)

## Como o problema de responsividade foi resolvido?

- A responsividade do site foi implementada utilizando regras **Media Queries no CSS**, permitindo adaptar o layout para diferentes tamanhos de tela.
